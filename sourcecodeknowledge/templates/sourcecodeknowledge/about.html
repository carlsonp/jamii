Frequently Asked Questions (FAQ):

Where did the name Jamii come from?
Jamii means "community" in Swahili.

What software libraries was Jamii built on?
Some images are from the Tango Desktop Project licensed under the public domain.
Built using Django, typeahead.js, DataTables, Bootstrap, jQuery, D3.js, jqPlot, and Neo4j.
Profile images are pulled from Gravatar.org.

Who's working on this project?
Patrick Carlson is a PhD candidate in Human-Computer Interaction at Iowa State University doing research on Free and Open Source communities.

I can't find my name or my name is incorrect.
My apologies! Jamii pulls data from multiple sources and some cleaning and formatting is done on the data. There are bound to be a few issues here and there both on the processing side as well as the data archives themselves.

Sure! Please see the Jamii Github page for the source code.
